## 실행 명령어

```
npm install
npm run build
npm run preview
npx wrangler deploy
```

preview 실패시 dist 폴더를 지우고 다시 build 할 것

## 내용 수정

- `src/data/products.ts` 파일에서 제품 category 수정
- `src/content/products` 폴더에서 제품 내용 수정
- `src/content/notice` 폴더에서 공지사항 수정
- `src/content/archive` 폴더에서 자료실 내용 수정

## 마크다운 작성 방법

`.md` 파일은 마크다운(markdown) 형식으로 작성함.
